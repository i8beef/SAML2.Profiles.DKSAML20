SAML2.Profiles.DKSAML20
=======================
SAML2.Profiles.DKSAML20 is an extension to the SAML2 library enabling custom profile validations for the Danish government.

Latest Version
==============
The quickest way to get the latest release of SAML2.Profiles.DKSAML20 is to add it to your project using NuGet (http://www.nuget.org/packages/SAML2.Profiles.DKSAML20).

Alternatively binaries are available from CodePlex at https://saml2profilesdksaml20.codeplex.com/.

Documentation
=============
The latest documentation for SAML2.Profiles.DKSAML20, including configuration instructions, bug reports, and discussions, will always be available on CodePlex.

https://saml2profilesdksaml20.codeplex.com/documentation.
